data-preprocessing-notes

결측치 처리, Label Encoding, One-Hot Encoding, SMOTE 기반 불균형 데이터 처리 실습을 Jupyter Notebook 형태로 정리한 프로젝트입니다.

Python Pandas SMOTE Machine Learning

프로젝트 개요

이 프로젝트는 Python(Pandas)을 기반으로 실제 데이터 전처리 실습을 진행하며 결측치 처리 → 인코딩 → 스케일링 → 데이터 불균형 해결(SMOTE) 흐름을 정리한 노트입니다. 각 전처리 기법의 특징과 주의점도 함께 기록하여 실무 적용 시 참고할 수 있도록 구성했습니다.

핵심 전처리 코드 예시

1) 결측치 처리 (fillna)
import pandas as pd

df = pd.read_csv("dataset/sample.csv")

# 평균값으로 결측치 대체
df["age"] = df["age"].fillna(df["age"].mean())

# 최빈값으로 결측치 대체
df["gender"] = df["gender"].fillna(df["gender"].mode()[0])
2) Label Encoding / One-Hot Encoding
from sklearn.preprocessing import LabelEncoder

le = LabelEncoder()
df["category_le"] = le.fit_transform(df["category"])

# One-Hot Encoding
df = pd.get_dummies(df, columns=["category"], drop_first=True)
3) 불균형 데이터 처리 (SMOTE)
from imblearn.over_sampling import SMOTE

smote = SMOTE(random_state=42)
X_resampled, y_resampled = smote.fit_resample(X, y)
