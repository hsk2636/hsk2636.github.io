통합 품질 정보 SQL 분석

quality-sql-analysis

LOT별/버전별 품질 데이터를 Oracle SQL로 집계하고, TOTAL ROW 생성 패턴까지 정리한 품질 지표 분석 프로젝트입니다.

Oracle SQL PL/SQL Manufacturing IT

프로젝트 개요

이 프로젝트는 제조 품질 시스템의 데이터를 대상으로 GOOD/BAD/M2 집계, 버전(REV)별 품질 현황, 월별 품질 현황 + TOTAL ROW 생성 패턴을 정리한 SQL 모음입니다.

파일 구성

```text quality-sql-analysis/ ├─ 01_quality_summary.sql # 기본 품질 집계 쿼리 ├─ 02_quality_version.sql # 버전(REV)별 GOOD/BAD 집계 ├─ 03_quality_total_row.sql # TOTAL ROW 생성용 SQL 패턴 ├─ 04_monthly_quality.sql # 월별 품질 현황 집계 └─ README.md # 프로젝트 소개 문서 ```

핵심 SQL 패턴 예시

1) LOT/버전별 기본 품질 집계
SELECT lot_no, rev_no, SUM(good_qty) AS good_qty, SUM(bad_qty) AS bad_qty, ROUND( CASE WHEN (good_qty + bad_qty) = 0 THEN 0 ELSE bad_qty / (good_qty + bad_qty) * 100 END, 2 ) AS defect_rate FROM tb_quality_result GROUP BY lot_no, rev_no ORDER BY lot_no, rev_no;
2) ROLLUP + TOTAL ROW 생성
SELECT CASE WHEN GROUPING(order_month) = 1 THEN 'TOTAL' ELSE order_month END AS order_month, SUM(defect_qty) AS total_defect_qty FROM tb_monthly_quality GROUP BY ROLLUP (order_month) ORDER BY CASE WHEN order_month = 'TOTAL' THEN 9999 ELSE TO_NUMBER(order_month) END;

위 패턴은 월별 집계를 하면서 마지막 행에 TOTAL을 추가해야 하는 리포트·대시보드에서 그대로 재사용할 수 있는 구조입니다.

활용 포인트

  • NVL / DECODE / ROLLUP / GROUPING 을 조합한 품질 리포트 SQL 패턴
  • LOT·REV 기준의 상세 집계와 월 단위 집계를 동시에 다루는 예제
  • 향후 PL/SQL 프로시저 안에서 재사용 가능한 집계 쿼리 구조
