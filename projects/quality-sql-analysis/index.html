통합 품질 정보 SQL 분석

quality-sql-analysis

LOT별/버전별 품질 데이터를 Oracle SQL로 집계하고, TOTAL ROW 생성 패턴까지 정리한 품질 지표 분석 프로젝트입니다.

Oracle SQL PL/SQL Manufacturing IT

프로젝트 개요

이 프로젝트는 제조 품질 시스템의 데이터를 대상으로 GOOD/BAD/M2 집계, 버전(REV)별 품질 현황, 월별 품질 현황 + TOTAL ROW 생성 패턴을 정리한 SQL 모음입니다. 보고서·대시보드에서 자주 사용하는 집계 패턴들을 한 곳에 모으는 것이 목표입니다.

핵심 SQL 패턴 예시

1) LOT/버전별 기본 품질 집계

품질 결과 테이블에서 LOT·REV 단위로 GOOD/BAD 수량과 불량률을 계산하는 예시입니다. 

SELECT
    lot_no,
    rev_no,
    SUM(good_qty) AS good_qty,
    SUM(bad_qty)  AS bad_qty,
    ROUND(
        CASE
            WHEN (SUM(good_qty) + SUM(bad_qty)) = 0 THEN 0
            ELSE SUM(bad_qty) / (SUM(good_qty) + SUM(bad_qty)) * 100
        END,
        2
    ) AS defect_rate
FROM tb_quality_result
GROUP BY
    lot_no,
    rev_no
ORDER BY
    lot_no,
    rev_no;

LOT/REV 단위의 기본 집계 패턴으로, 이후 공정/고객/제품별 Drill-down 보고서의 기초 데이터 소스로 활용할 수 있습니다.

2) ROLLUP + TOTAL ROW 생성

월별 집계와 함께 마지막 행에 TOTAL을 추가하는 리포트용 SQL 패턴입니다. 

SELECT
    CASE
        WHEN GROUPING(order_month) = 1 THEN 'TOTAL'
        ELSE order_month
    END AS order_month,
    SUM(defect_qty) AS total_defect_qty
FROM tb_monthly_quality
GROUP BY ROLLUP (order_month)
ORDER BY
    CASE
        WHEN order_month = 'TOTAL' THEN 9999
        ELSE TO_NUMBER(order_month)
    END;

GROUP BY ROLLUP + GROUPING() 조합을 통해 월별 행 + 전체 합계 행을 동시에 만들 수 있습니다. 엑셀 피벗이나 BI툴에서 보는 형태를 그대로 SQL 레벨에서 구현하는 느낌입니다.

활용 포인트

  • NVL / DECODE / ROLLUP / GROUPING 을 조합한 품질 리포트 SQL 패턴 학습
  • LOT·REV 단위 상세 집계와 월 단위 집계를 동시에 다루는 예시
  • 향후 PL/SQL 프로시저 안에서 재사용 가능한 집계 쿼리 구조 설계 연습
  • 제조 품질 KPI(불량률, 불량 수량 등) 계산 로직을 문서화하는 연습
